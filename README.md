## vuex-demo1
计数器
1. 使用mutations,store用来接收
2. 使用actions实现异步操作
3. 使用getter实现缓存（类似于计算属性）

## vuex-demo2

1. 感觉极客时间的代码很精炼，自己对代码的阅读存在困难
2. 需要熟练掌握ES6，包括vuex的一些封装操作

## vuex-shopcart
购物车---个人开发
1. 添加商品到购物车，实现归类，并且计算出总价
2. 实现商品的余量处理，随着商品的添加和删除进行动态更改
3. 实现同类物品批量删除的功能
4. 实现清空购物车的功能
